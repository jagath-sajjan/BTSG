# BTSG Demo Quick Reference Card

**⏱️ Total Time: 90 seconds**

---

## 🎬 Commands (Copy-Paste Ready)

```bash
# 1. SCAN (15s)
btsg scan .

# 2. EXPLAIN (20s)
btsg explain --finding-id 1

# 3. FIX (20s)
btsg fix --vuln-id 1 --preview

# 4. REPORT (15s)
btsg report --format html --output security-report.html

# 5. WATCH (5s - bonus)
btsg watch --verbose
```
